# 2. JK-Flip-flopの特徴
JK-Flip-flop (JK-FF)は、-1, 0, 1を記憶する。
JS-FFは、演算結果の $\overline{Q}, Q, -1, 0, 1$ が得られる。
各入力値(J,K)を変えることで、SetReset-FF, Toggle-FF, Delay-FFを再現できる。

______________________________________________________________________________
# 3. 特性表

JK-FFの特性表を以下のように定める。
$Q^+$ は、次状態とする。

|  J   |  K   |  $Q^+$ |
| ---- | ---- | ----   |
|  -1  |  -1  | $Q$ |
|  -1  |   0  | -1|
|  -1  |   1  | $\overline{Q}$ |
|   0  |  -1  | 0 |
|   0  |   0  | $Q$ |
|   0  |   1  | 0 |
|   1  |  -1  | $Q$|
|   1  |   0  | 1 |
|   1  |   1  | $\overline{Q}$ |
